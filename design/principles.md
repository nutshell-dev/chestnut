# Clawforum 重构原则

## Philosophy

* Agent即目录
* Clawforum本质是上下文工程
* 控制系统复杂度是核心架构约束
* 分多个智能体、分子任务，目的是更高效地利用上下文窗口，而不是模拟人类组织架构的分工，各个claw的特长能力可以轻易复制过来
* 系统为智能体服务，提供基础设施和必要信息

## Design Principles

* 运行中的所有信息不丢失，状态可观察，中断可恢复，事后可审计
* 运行中产生的任何信息，未经显式设计决策，不得丢弃或静默忽略
* 用户可以观察运行过程中的所有状态
* 运行中断即从最后一次完整LLM调用恢复状态并继续
* 事后仅凭日志和记录，能完整重建任一时刻的运行状态和决策链路
* 磁盘即权威，内存可派生但不能为权威（运行时句柄从磁盘信息重建）
* 智能体是决策主体，执行走受约束路径；创建子代理后不阻塞，结果异步返回
* 智能体创建的临时子代理完全继承调用方的OS资源访问权限（"权限"指 sandbox 边界 / 可访问的文件路径集合 + 命令准入 / 不指工具入口数 / 子代理可有比调用方更多工具入口（如 edit/multi_edit）只要边界一致）
* 系统在智能体需要决策时交付相关信息
  * 恰好在需要的时候提供，避免智能体上下文负担。采用事件驱动
* 多个claw智能体的信息不应当隔绝，允许互相访问
* motion为clawforum中一个特殊的claw，对其他claw有单向访问权
* CLI 是claw和motion的唯一对外入口，分布式场景下可替换为远程调用
* clawforum 支持分布式部署和跨 OS 平台（未来演进方向）

## Module Logic Principles

架构的核心是控制系统复杂度，耦合是系统复杂度扩散的路径，通过管理依赖来控制耦合。

模块逻辑原则要足够泛化，能覆盖未预见的情况，而不是对已知情况的枚举描述

* 一个模块封装一组独立可变的职责，每种职责只归一个模块
* 模块为自己的业务语义负责，需要业务语义的动作由该模块发起
* 每种资源只归属唯一模块，其他模块通过该模块的对外入口间接访问
* 持久化一切信息到磁盘（运行时句柄从磁盘信息重建）
* 模块的依赖单向，禁止双向/循环，底层模块不预设上层模块语义
* 模块的依赖结构稳定：运行时模块间依赖关系不可变
* 模块的耦合界面稳定：对外表面不随内部实现或外部模块增减而变化
* 模块的耦合界面最小：对外表面只暴露必要的最小集合，跨边界只传必要信息
* 不可消除的耦合应显式表达，优先表达为让编译器检查
* 发现模块设计架构、职责归属不合理时，停下来讨论模块重构
* 模块边界设计和实际依赖对不上时，停下来讨论模块边界重构

## Path Principles

Design Principles 与 Module Logic Principles 描述系统的应然,Path Principles 指导从实然到达应然的路径,将由实践检验和改进。

Path Principles 是 Module Logic Principles 的动态镜像（2026-04-22 phase219 整理）：
- M#1（职责独立可变 / 唯一归属）↔ Path #3（语义原子 / 单位有边界）
- M#2（业务语义归属 / 动作由归属方发起）↔ Path #7（归属原则）
- M#5（依赖单向 / 信息流单向）↔ Path #1（规划基于事实）
- M#7（耦合界面稳定 / 稳定性保证）↔ Path #4（可回滚 / 破坏性论证）
- M#9（不可消除耦合显式）↔ Path #2（差距显式登记）
- M#10/#11（发现问题停下）↔ Path #6（冲突立即中断）

* 路径规划要基于规划时刻的事实,注意实施过程中实然的变化
* 实然和应然的差距必须显式登记,在路径规划时参考
* 实施过程中每次只做**语义原子的变更单元**
* 变更必须可回滚,破坏性改动须显式论证
* 变更完成后必须复盘,由实践反馈规则
* 实施中发现原则冲突要立即中断,冲突调整优先于强行推进
* 实然到达应然有诸多路径，要选择降低总难度的路径（步骤间有相互作用，总难度不等于各步骤成本简单相加）

## Coding Principles

代码的目标是正确实现上层设计。编码原则服务于这个目标,降低写出不正确实现的风险。

编码原则要足够泛化,能覆盖未预见的情况,而不是对已知情况的枚举描述。

**结构**

* 减少代码间的依赖：改一处不应连带改多处
* 每一层抽象只隐藏调用方不需要关心的细节

**数据**

* 让状态的变化可预测：可变状态应有唯一且明确的管理者

**错误**

* 预期失败由调用方显式处理,不可预期失败暴露而非吞没

**命名**

* 名字准确反映意图,同一概念用同一名字

**测试**

* 测试验证行为契约,让代码经历从未走过的路径
